## 5. Proof of Mutation (PoM)

### 5.1 Consensus Mechanism

Proof of Mutation (PoM) is BIOCOIN's consensus algorithm that replaces Proof of Work:

**Traditional PoW (Bitcoin)**:
```
1. Take block data
2. Hash with nonce
3. Check if hash starts with N zeros
4. Repeat until valid
```

**Proof of Mutation (BIOCOIN)**:
```
1. Take previous block's DNA sequence
2. Mutate DNA following biological rules
3. Hash mutated DNA
4. Check if hash starts with N zeros
5. Repeat until valid
```

### 5.2 Mining Process

#### Step 1: Get Previous DNA
```
Miner requests latest block from node
Extracts DNA sequence from previous block
Example: "CCCAGCAC"
```

#### Step 2: Mutate DNA
```
Apply biological mutations:
- Transition: C â†’ T (70% chance)
- Transversion: C â†’ A (25% chance)
- Insertion: Add base (4% chance)
- Deletion: Remove base (1% chance)

Example mutation: "CCCAGCAC" â†’ "CCTAGCAC"
```

#### Step 3: Hash Mutation
```
Hash mutated DNA: SHA256("CCTAGCAC")
Result: "0000abc123..." (if meets difficulty)
```

#### Step 4: Validate
```
Check if hash starts with required zeros (difficulty)
If valid â†’ Create block
If invalid â†’ Try another mutation
```

#### Step 5: Submit Block
```
Miner creates block with:
- Mutated DNA sequence
- Transaction data
- Block metadata
- Mutation records

Submits to node for validation
```
